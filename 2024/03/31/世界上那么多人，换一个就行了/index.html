世界上那么多人，换一个就行了

Posted at 2024-03-31 人  朋友 三观 

一月学习状态十分差，到了二月觉得再这样下去，找到一份码农工作十分渺茫。想起在看到网友推荐过一个免费找导师的网站，我想着看看能不能找到人指点一下我的学习路径。

上面 front-end 的导师不算很多，坐标德国的女性更少，于是我只能开始找男导师。第一个男导师是德国人，在我写了介绍之后，他马上在后台给我回答了一个问题。然后说他有好方法可以教我学德语，他在前妻（台湾人，会说普通话和英语）身上试验过。

我看到他这样说有点期待，同时有点奇怪：既然你有好方法，你也回复我了，那你可以在这里把方法说清楚的嘛。但是我当时觉得他可能有他的想法，我们也还没见，就等见了面再说吧。

见面的时候，他说不会德语找工作确实有点麻烦，A1 是真不够，最起码得假装有 A2 吧（但是之前见的印度导师说 B1 也不够）。他可以带我练练德语，再帮我细化学 JS 的计划。

他查了一些实习的薪资，说很低，这个应该申请不了工签。我也不清楚工签薪资要求，去查了一下。华人论坛里有人提到看行业，如果低于行业标准水平，可能会被卡。而且最好是工业或商业。忽然觉得，可能我找份超市收银员的活，未必能换到工签。

因为我申请的是一小时面谈，但是我们超时了。结束前，这个男的最后问我，“just curious，你为什么会剪这么短的头发？”我：因为方便啊。他：我只是好奇，因为很少见女生剪这样的发型。我当时就觉得很奇怪，你们德国人不是很多男的都爱留长头发吗？各种奇怪打扮都有吧，你为何好奇我的头发呢？但是我对他还是没上面戒备心，又见他之前说得那么好，就和他说我这个发型在亚洲女性里确实比较少见。然后说了这样的发型容易受到什么歧视，所以我才更想留在德国。

结束之后，他给我一个第二天半小时的 session，说带我练习德语；还建立了只有我们可见的 GitHub repository。

第二天，他道歉说他前一晚太累了（不是生病），所以就早睡了，没有给我留 js 小作业。其实我忘了他前一天说给我留作业的事了。随后，他和我聊了一个多小时，我问了一些德语问题。他建议我多听收音机，建立语感。我说我没有收音机，只有播客。他说他还是倾向于收音机，就放着，当背景。他前妻就是那样学的，多听，看电视什么的，然后开始说。我当时想着，播客+音乐播放器，不就有收音机的功能了吗，他为什么一直强调收音机？但是我没说出来。有时候我犯错的时候，他说看到我犯这样的错，又想起了他前妻。当然，不是因为我长得像她，只是因为大家入门都犯这样的错。我当时觉得挺不舒服的，因为我觉得他一直提起他前妻，感觉特别怪。再加上前一天问我发型的时，感觉就更怪了。

而且这个大哥老是说我有些发音不好，但是我觉得我发音很好。他还说每个地方的德国人口音都不一样，他在南部就怎样；说不知道你在哪里，那里的人是什么口音。当时我觉得他很想知道我在哪里，但是我觉得不想让他知道，所以我就没接这个话题。

他还和我说了一下他想给我找实习，可以带薪学习，虽然钱不多，但是可以学到知识和争取多点时间。但是他目前没找到合适的，所以想在未来两个月内带我完成 A2。他也可以给我写介绍信，说我们 mentor 期间学了什么，但是他也不确定有多大作用。他十来年前就是参加了这个培训，然后找到工作的。现在也会在一些机构里教人编程。我终于反应过来他说的是 ausbildung（双元制培训），就告诉他这个要求 25 岁以下，我已经超龄了。他说，哎呀，你看着好年轻啊，你该不会比我大吧？我：有可能呀。他说我 35 岁。我说那我比你小一些。他说他有个朋友 30 岁了还是去参加了这个培训，我就没再接话了。结束前他说他明天有空，于是又约了一个半小时 session。

其实我觉得和他聊也挺累的，而且有用信息并不是很多。但是想着人家那么热心，我就当学习吧。就算没学到德语和编程，就当练练英语口语，毕竟我口语太差了。

第三天还是约的五点，是他自己说他那天有空，定的时间。但是到了那天五点，他没出现，也没邮件通知我（我不确定他是否有我的邮箱）。于是我登录了那个导师网站，最搞笑的是，网站显示他五点零五分才离开的。

我也懒得问他为什么不出现，为什么不留意，反正我觉得连着和他聊天也很累。也没有很想问的问题，我还查了一下我昨天出现的错误，知道了正确的答案。通过后台，我才发现原来前一天聊完之后，他给我布置了 JS 小作业，但是邮箱和 Github 没提醒我有消息，所以我那天才看到。如果他和我面谈，我应该也不会打开网页后台，也就看不到那个小作业。但老实说，我没懂那个作业是什么。

那天晚上十点多还是十一点多，我收到邮件通知，他在后台私信我了。但是私信只能登陆网站才能看到，还会显示你最后在线时间。我并不想直面他，就没打开网站。第二天早上，我打开后台，看到他昨晚八点多给我信息说他太抑郁所以不能起床。说可以 discord 聊，他经常在那。我们可以另外约个时间。

我就感觉，如果他真是抑郁到起不来，这行动力和精神头都太强了。他的预约网页，每天早上六点多开始有几个位置，下午三四点开始有几个位置，每天不间断。这是我见过开发最多预约的导师了。也不知道他是在逼自己还是啥。但是真抑郁逼最近也没用啊。但是我当时还是给他留言说第二天下午五点有空。

之后我还是觉得很不对劲，就和朋友说了这件事。她说换以前，她也会再给这个人一次机会。但是从她职场经验看，没有任何人会无缘无故对另一个人这么好。她很担心他会不会是杀人犯。就算我有再多佐证，但是依然觉得奇怪，所以要相信自己的直觉。他已经鸽我两次了，没必要给同一个人很多次（伤害你的）机会，世界上那么多人，换一个就行了。我：那我已经和他说了什么时候有空了，明天要怎么拒绝他？她：他之前鸽你的借口都很随意，你也随意找一个行了。我：好的。

“世界上那么多人，换一个就行了。”我感觉，可以用在很多事情上。做到了，烦心事少很多。
